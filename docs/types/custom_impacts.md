# Custom Impacts

Lunali adds 3 custom impacts for you to use. These impacts are Extreme (4), Boss (5), and Custom (6)

This isn't really rocket science or a major break through and will probably have niche uses, but if you want to see what the impacts look like you can head to [Custom Origin Backgrounds](origin_backgrounds.md) and look at the template, the impacts are also on the default origin GUI, so no need to set a custom background.

```JSON
{
  "name": "Lunali",
  "description": "Test origin for Lunali powers",
  "impact": 6,
  "icon": {
    "id": "origins:orb_of_origin"
  },
  "powers": [
    "lunali:test"
  ]
}

```

This example gives the Lunali origin the "Custom" Impact
